# Connect Four
Working but still in development...


Online connect four allowing users to play online through entering the same room code. 
Made with a Flask server using Flask Socket IO and a React with Vite. 

To run the Flask server:
1. Create a virtual environment `python -m venv venv`
2. Install the requirements `pip install -r requirements.txt`
3. Start the server `python run.py`

To run the React client:
1. Change directory `cd frontend`
2. Install the dependencies `npm install`
3. Run the client `npm run dev`
